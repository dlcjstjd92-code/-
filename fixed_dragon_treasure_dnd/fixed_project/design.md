# 용이 감춘 보물 그리고 사냥꾼 — 디자인 계획

## 앱 개요
탈라즘제국을 배경으로 한 AI 기반 텍스트 D&D RPG. 2D6 주사위 시스템, 몰입형 내면 나레이션, NPC 상호작용, 대서사 이벤트, 일대기 저장 기능을 포함한 판타지 게임.

---

## 화면 목록

### 1. 스플래시/인트로 화면 (`/intro`)
- 앱 로고 + 게임 제목 표시
- 인트로 텍스트 애니메이션 (타이핑 효과)
- "이야기속으로 들어가시겠습니까?" 버튼
- 이어하기 버튼 (저장된 게임 있을 경우)

### 2. 이름 입력 화면 (`/name-input`)
- 베토 스승 첫 만남 씬
- 이름 입력 텍스트 필드
- 확인 버튼

### 3. 게임 메인 화면 (`/game`)
- **상단 상태바**: 날짜/시간, 위치
- **시스템 로그 패널**: 주사위 결과, 변화 사항
- **내면 나레이션 영역**: 소설형 서술 (스크롤 가능)
- **선택지 패널**: 직접 입력 + 추천 선택지 2개 + 지식판정 + 관찰판정
- **하단 상태바**: 보유물품, 현재 상황

### 4. 일대기 화면 (`/diary`)
- 날짜별 일대기 목록
- 각 일대기 소설 형식 표시
- 게임 종료 시 전체 일대기 읽기 모드

### 5. 설정/메뉴 화면 (`/menu`)
- 게임 저장 (일대기 기록)
- 능력치 확인
- 보유 물품 상세
- 게임 종료

## 핵심 사용자 플로우

### 신규 게임 시작
1. 스플래시 → 인트로 텍스트 표시
2. "예" 선택 → 이름 입력 화면
3. 이름 입력 → 시작 이벤트 (베토 스승 꿈 씬)
4. 게임 메인 화면 진입

### 게임 진행
1. AI가 나레이션 + 선택지 생성
2. 플레이어가 선택지 선택 또는 직접 입력
3. 주사위 굴림 애니메이션 표시
4. AI가 결과 나레이션 생성
5. 상태/아이템 변화 반영

### 이어하기
1. 스플래시 → 이어하기 버튼
2. 마지막 일대기 기반으로 게임 상태 복원
3. 게임 메인 화면 진입

### 수면/일대기 저장
1. 수면 선택 → AI가 하루 요약 생성
2. 일대기에 소설 형식으로 기록
3. 게임 상태 저장

## 타이포그래피
- **나레이션 텍스트**: 세리프 느낌의 폰트, 16sp, 줄간격 1.6
- **선택지 버튼**: 14sp, 굵게
- **상태바 텍스트**: 12sp, 모노스페이스 느낌
- **주사위 결과**: 18sp, 황금빛 강조

## 애니메이션
- 나레이션 텍스트: 타이핑 효과 (스트리밍)
- 주사위: 굴림 애니메이션 (간단한 숫자 변환)
- 선택지: 페이드인 효과
- 화면 전환: 페이드

## 주사위 시각화
- 나무주사위: 갈색 텍스트
- 쇠주사위: 회색 텍스트
- 은주사위: 은색 텍스트
- 금주사위: 황금색 텍스트
- 흑광주사위: 흑광 보라빛 텍스트
